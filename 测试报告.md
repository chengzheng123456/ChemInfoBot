## 使用步骤

### 第一步：配置邮箱（必须）

编辑 `config.py` 文件:

```python
EMAIL_CONFIG = {
    "smtp_server": "smtp.163.com",
    "smtp_port": 465,
    "sender_email": "你的邮箱@163.com",      # 修改此处
    "sender_password": "你的SMTP授权码",      # 修改此处
    "recipient_email": "cz9721@163.com",    # 已配置正确
    "use_ssl": True,
}
```

**获取SMTP授权码方法:**
1. 登录163邮箱网页版
2. 点击 "设置" → "POP3/SMTP/IMAP"
3. 开启 "SMTP服务"
4. 按照提示获取授权码（不是邮箱密码）

### 第二步：启动Web管理界面

双击 `启动Web界面.bat` 或在CMD中运行:

```bash
cd D:\Backup\Documents\ChemInfoBot
python web_app.py
```

访问: http://localhost:5000

### 第三步：启动定时任务

双击 `启动定时任务.bat` 或在CMD中运行:

```bash
python scheduler.py
```

选择 `1. 后台模式 (持续运行)`
