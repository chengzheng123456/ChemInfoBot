```python
EMAIL_CONFIG = {
```

2. **获取SMTP授权码**:
   - 登录 [163邮箱](https://mail.163.com)
   - 设置 → POP3/SMTP/IMAP
   - 开启 SMTP 服务
   - 获取授权码（16位字符串）

3. **测试邮件**:
   ```bash
   python -c "from email_sender import email_sender; email_sender.test_connection()"
   ```

## 🚨 常见问题

### Q1: 爬虫没有自动运行？
**A**: 确保调度器已启动：
```bash
# 检查Python进程
tasklist | findstr python

# 如果没有，重新启动
python start_scheduler.py
```

### Q2: 邮件发送失败？
**A**: 检查邮箱配置：
1. 确认 `config.py` 中的邮箱和授权码正确
2. 确认163邮箱已开启SMTP服务
3. 测试连接: `python -c "from email_sender import email_sender; email_sender.test_connection()"`

### Q3: 如何修改推送时间？
**A**: 编辑 `config.py`:
```python
SCHEDULER_CONFIG = {
    "daily_push_time": "08:30",  # ← 修改为所需时间，如 "09:00"
    "timezone": "Asia/Shanghai",
}
```
